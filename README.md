# AI 创作工作台

一个纯前端的 AI 创作工具，支持图片分割和 AI 图像/视频生成。

🔗 **在线访问**：https://你的用户名.github.io/ai-studio-workbench/

## 功能特性

### ✂️ 图片分割（完全免费，无需 API Key）
- 支持 3×3 / 4×4 / 5×5 均等切割
- 支持 16:9 和 9:16 图片比例
- 输出尺寸可选：保持原始 / 1080px / 800px / 640px
- 完全本地运行，不上传任何数据

### 🎨 AI 图像生成
- 接入阿里云百炼平台
- 支持多种图像模型

### 🎬 AI 视频生成
- **Seedance 2.0**（火山引擎）：图生视频、首尾帧、多图参考
- **可灵 Kling v3.0**（阿里云百炼）：文生视频、图生视频
- **MiniMax 海螺 Hailuo 2.3**：文生视频、首尾帧、运镜指令

## 使用说明

### 图片分割（立即可用）
1. 打开网页，点击顶部「✂️ 图片分割」标签
2. 上传图片（支持 JPG/PNG/WebP，最大 50MB）
3. 选择分割方式（3×3 / 4×4 / 5×5）
4. 点击「开始分割」，右侧预览所有格子
5. 点击单张下载，或使用「批量下载」

### AI 生成功能（需要 API Key）

点击右上角「⚙️ API 配置」，填入以下 Key：

| 平台 | 申请地址 | 用途 |
|------|----------|------|
| 阿里云 DashScope | https://dashscope.aliyun.com/ | 图像生成 + 可灵视频 |
| 火山引擎 | https://console.volcengine.com/ | Seedance 视频 |
| MiniMax | https://www.minimaxi.com/ | 海螺视频 |

**注意**：Key 仅保存在浏览器本地，不会上传到任何服务器。

## 本地使用

直接在浏览器中打开 `index.html` 文件即可使用，无需安装任何依赖。

## 技术说明

- 纯前端实现，使用 Vue 3 + Tailwind CSS
- 图片分割使用浏览器 Canvas API，完全本地处理
- AI 功能通过各平台官方 API 调用
